Flujo del usuario

La web solo registra la solicitud. El procesamiento operativo lo realiza el robot.

1
Crear campaña

El usuario define nombre, tipo de proceso y responsable.

2
Seleccionar filtros

Se eligen los criterios de segmentación para Raw_Call, Raw_CC o campaña personalizada.

3
Confirmar solicitud

La solicitud queda pendiente para que el robot la tome y ejecute el flujo automático.

4
Consultar estado

El usuario ve avance, resultado, archivo generado o error desde el historial.
